# 🏦 Smart Banking System

A fully functional Banking System project built in **C++** 
with a complete **HTML/CSS/JavaScript** frontend.

## 📌 Features

- ✅ Create New Bank Account
- ✅ Login with PIN (locks after 3 wrong tries)
- ✅ Deposit Money
- ✅ Withdraw Money
- ✅ Transfer Money to Another Account
- ✅ View Account Details
- ✅ Fixed Deposit (FD) Calculator
- ✅ Loan Eligibility Checker
- ✅ Admin Panel (view, delete, unlock accounts)
- ✅ Data saves automatically


## 🛠️ Built With

- C++ (core logic, file handling)
- HTML & CSS (frontend UI)
- JavaScript (browser-side logic)
- localStorage (data storage in browser)

## 🚀 How to Run

### Run C++ version (Terminal):
```bash
g++ banking_system_final_cpp.cpp -o bank
./bank
```

### Run HTML version (Browser):
Just open `SmartBankingSystem.html` in any browser. No setup needed!
